## About the project
A simple Node.js web app that serves static HTML file content.

## Dependencies
1. Node.js

## Installation
- 1 - Clone this repo:

```
git clone
```

- 2 - Copy/paste static website in /public

* 3 - Run this command:

```
npm install
```

- 4 Run this command:

```
npm start
```

- 4 Point your browser to:

```
http://localhost:3000
```
The app should be at http://localhost:3000.
